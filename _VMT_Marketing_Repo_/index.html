Häufige Fragen

Ist die Software wirklich kostenlos?

Ja, das VereinsManagementTool ist ein Open-Source-Projekt von David Schuchert und steht allen Vereinen dauerhaft kostenfrei zur Verfügung.

Wo werden meine Daten gespeichert?

Die Anwendung wird auf Ihrem eigenen Server installiert. Damit behalten Sie die volle Souveränität über alle Mitgliederdaten – perfekt für den Datenschutz.

Benötige ich IT-vorkenntnisse?

Die Bedienung ist so einfach wie Online-Banking gestaltet. Für die einmalige Installation stellen wir eine ausführliche Dokumentation bereit.
