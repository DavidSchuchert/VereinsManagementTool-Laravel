# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.0] - 2026-04-14

### Added
- **Docker Support**: Completely modernized Docker setup with live development support.
- **GitHub Professionalization**: Added standard community files (`CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`).
- **Automation**: Added GitHub Actions for CI (testing with PHP 8.4) and Dependabot for dependency updates.
- **Issue Templates**: Added structured templates for Bug Reports and Feature Requests.
- **Documentation**: Added Table of Contents and Screenshots to README.md.

### Changed
- Improved Docker performance by caching dependencies and optimizing the entrypoint.
- Updated PHP requirement to 8.4 for Docker environment.

## [2.1.0] - 2026-03-05
- Internal release with initial Livewire 3 integration.
